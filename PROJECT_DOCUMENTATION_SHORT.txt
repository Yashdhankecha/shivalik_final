PROJECT OVERVIEW
----------------
Shivalik Community Management Platform is a comprehensive full-stack web 
application designed to streamline community management for real estate 
communities. It provides a centralized platform for residents, managers, and 
administrators to interact, manage events, buy/sell items, share updates, and 
maintain a thriving community ecosystem.

KEY FEATURES
------------
1. Community Management
   - Create and manage multiple communities
   - Join request system with approval workflow
   - Member management and role assignment

2. Pulses (Social Feed)
   - Create and share community posts
   - Like and comment functionality
   - Manager approval workflow

3. Event Management
   - Create and manage community events
   - QR code-based registration
   - Digital attendance tracking

4. Marketplace
   - Buy/sell property listings
   - In-app chat between buyers and sellers
   - Listing approval workflow

5. Moderation Dashboard
   - Overview of all pending items
   - Approve/reject join requests
   - Moderate posts and listings
   - Quick stats and counters

6. Service Directory
   - Curated directory of service providers
   - Search and filter functionality

TECH STACK
----------
Frontend:
- React 18.3.1 with TypeScript
- Vite 6.4.1
- Tailwind CSS 3.4.11
- Redux Toolkit + Redux Saga
- Radix UI components

Backend:
- Node.js 20.18.0
- Express.js 4.21.1
- MongoDB 8.7.1 (Mongoose)
- JWT Authentication
- Express-fileupload

QUICK START
-----------
1. Backend Setup:
   cd server/services/community_services
   npm install
   npm run start:dev

2. Frontend Setup:
   cd client
   npm install
   npm run dev

3. Access:
   Frontend: http://localhost:8080
   Backend: http://localhost:11001

USER ROLES
----------
- Admin: Full system access, manage all communities
- Manager: Manage assigned communities, moderate content
- User: Join communities, create posts, register for events
- Guest: View landing page, browse communities

MAIN MODULES
------------
1. Authentication System - Registration, login, OTP verification
2. Community Management - Create, join, manage communities
3. Pulses Module - Social feed with posts, likes, comments
4. Events Module - Event creation, registration, QR attendance
5. Marketplace Module - Buy/sell listings with chat
6. Directory Module - Service provider directory
7. Moderation Dashboard - Content approval workflow
8. Admin Panel - System-wide management
9. Manager Panel - Community-specific management

API ENDPOINTS
------------
Base URL: http://localhost:11001/api/v1

Key Endpoints:
- POST /auth/register - User registration
- POST /auth/login - User login
- GET /community/communities - Get all communities
- POST /community/pulses/create - Create post
- POST /community/events/create - Create event
- POST /community/marketplace/listing/create - Create listing
- GET /manager/moderation-dashboard/:communityId - Moderation dashboard
